    # 价格说明

    # 概述

    裸金属服务器（Bare Metal Server）计费方式为包年包月付费。不支持按量付费。其中套餐会有专属优惠价格。对于使用者，推荐套餐购买。

    # 计费项

    BMS对您选择的裸金属服务器、带宽（可选）、云硬盘（可选）、弹性IP（可选）、共享带宽组（可选）、快照/镜像存储包（可选）收费。

    # 变更配置

    扩容存储空间：如果您的业务需求增大，可以通过扩容或者增加裸金属服务器挂载的云硬盘来增加您的存储空间。这部分存储空间将按照您选择的云硬盘计费方式计费。
    升级配置：如果性能无法满足您的业务需求，可以通过IPC升级的方式一键更换迁移新的平台。升级需要补交原机器计费的差价。
