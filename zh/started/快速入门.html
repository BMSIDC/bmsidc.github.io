# 快速入门

# 入门指引

本文档将以部署一台Web应用服务器为场景，帮助您学习如何创建和使用裸金属服务器。通过本文档，您可以学习到如何挑选合适的服务器，如何登录服务器，以及部署Nginx软件。

您将按以下流程学习如何使用裸金属服务器。

# 准备工作

在使用裸金属服务器前，您需要注册万维讯通会员帐号。

# 注册万维讯通

如果您已有一个万维讯通账号，请跳到下一个任务。如果您还没有万维讯通账号，请参考以下步骤创建。

  1. 打开万维讯通官网，单击“登录/注册”。
  2. 根据提示信息完成注册。
  3. 注册成功后，系统会自动跳转至您的个人信息界面。

# 步骤1：创建裸金属服务器

# 操作场景

您可以根据以下步骤使用管理控制台创建一个裸金属服务器实例。本教程以部署一台Web应用服务器为场景，旨在帮助您快速创建一个实例，因此不会涵盖所有可能的选项。

# 操作步骤

  1. 登录云服务器控制台

  2. 在左侧导航栏，选择“裸金属服务器 BMS”

  3. 单击上方的“创建”

  4. 完成参数配置

    • 选择机房和数据中心，如“Hong Kong- BMSIDC-Hong Kong”，可用区保持默认值。
      说明： 裸金属服务器创建完成后，不可更改区域和可用区。

    • 选择“机型”
      可供选择的规格由您所选择的区域和可用区决定。Web服务器以满足网页访问为主，不需要很强的计算能力。我们选择“BMS_D1528C16”。

    • 添加“磁盘”
      创建服务器会免费赠送128G的系统盘，可以根据自身需求，选择添加云盘。

    • 选择“镜像”
      选择“公共镜像”，“CentOS 7.6 64bit”。 系统自动分配随机登录密码，您也可以指定设置，建议采用复杂的密码方案。

    • 选择“网络”
      您第一次使用云服务时，系统将自动为您创建一个虚拟专有网络 VPC和物理NAT网关，保持默认值即可。

    • 选择“带宽”
      根据您的业务需要选择合适的线路。并购买带宽供裸金属服务器使用。

    • 添加“弹性公网IP”
      不使用弹性公网IP的裸金属服务器不能与互联网互通，仅可作为私有网络中部署业务或者集群所需裸金属服务器进行使用。

    • 设置“购买时长”
      范围：1天~3年。1天仅供测试使用，建议续费采用最低月付的方式。

    • 设置“购买数量”

  5. 点击 “立即购买”，确认规格无误后，再次点击 “立即购买”。创建订单。选择一种支付方式，单击“确认付款”。

  6. 执行结果
    一般需要10s~30s完成裸金属服务器创建。单击刷新按钮，裸金属服务器状态由“创建中”变为“运行中”，表示创建成功。

  7. 后续操作
    裸金属服务器作为Web服务器对外提供服务，还需要放通80、443端口以及ICMP协议，默认系统防火墙是关闭的，无需操作。对安全有要求的客户可以参看安全组设置。

# 步骤2：登录裸金属服务器

# 操作场景

创建裸金属服务器后，您可以通过多种方式进行登录。
本文介绍在管理控制台使用管理终端快速登录裸金属服务器。更多登录方式，请参阅Linux服务器登录方式说明或Windows服务器登录方式说明。

# 操作步骤

  1. 登录云服务器控制台
  2. 在左侧导航栏，选择“裸金属服务器”。
  3. 在裸金属服务器列表中，找到已创建的实例“bms-xxxxx”。在“更多操作”列，单击“远程桌面”。 开始建立连接，大约几秒后后进入系统界面。出现如下信息表示登录成功。
[root@Centos76-Yx4PdGGT ~] #

提示

默认会以创建时的默认登录信息进行连接，如果您有修改过，需要填写正确的登录信息。

# 步骤3：部署应用环境

  1. 安装及启动Nginx
    输入yum install nginx命令安装Nginx，当需要确认时输入“y”确认。
    出现下图表示Nginx安装成功。
Installed:
  nginx.x86_64 1:1.16.1-1.el7

Dependency Installed:
         ...................
         ...................

Complete!
[root@Centos76-Yx4PdGGT ~] #

  1. 输入systemctl start nginx.service启动Nginx服务。
    此处以CentOS 7.6 64bit为例，其他操作系统的Nginx启动命令请您自行查阅。

  2. 访问Web默认页面
    使用浏览器访问“http://裸金属服务器的弹性公网IP地址”，看到Nginx的欢迎页面即表示安装成功。
