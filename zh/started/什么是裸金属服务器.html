# 裸金属服务器介绍

裸金属服务器（Bare Metal Server，BMS）是一款兼具虚拟机弹性和物理机性能的计算类服务，为您和您的企业提供专属的云上物理服务器，为核心数据库、关键应用系统、高性能计算、大数据等业务提供卓越的计算性能以及数据安全。租户可灵活申请，按需使用。

裸金属服务器的开通是自助完成的，您只需要指定具体的服务器类型、镜像、所需要的网络配置等，即可在30s内获得所需的裸金属服务器。服务器供应、运维工作交由万维讯通，您可以专注于业务创新。

# 规格配置

云平台提供适用于不同业务场景的实例类型，不同类型的实例在CPU、内存容量、存储介质、存储容量、网卡数量等维度有不同的配置。 更多关于实例配置的介绍，请参考实例章节。

# 存储配置

支持从云硬盘启动，实现分钟级发放。

# 网络配置

支持虚拟私有云（VPC），为同一可用区内的裸金属服务器之间提供带宽不受限制的网络，内网带宽可达10Gb/s以上。 更多关于网络配置的介绍，请参考网络章节。

# 镜像配置

可以使用公共镜像、私有镜像以及共享镜像。

# 安全配置

保持和云服务器相同的安全组策略。
